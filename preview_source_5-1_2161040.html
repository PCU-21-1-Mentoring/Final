1. HTML5 문서구조의 기본 태그가 아닌 것은?

<br>

2. 다음 중 HTML 문서의 정보를 기술하기 위해 사용하는 태그는?

<meta>

3. 태그의 내용이 보이지 않도록 하는 속성은?

<hidden>

4. 웹 문서의 레이아웃(layout)을 결정하기 위해 사용하는 구조적 태그가 아닌 것은?

<position>

5. 웹 문서의 구조적 태그 중에서 ‘독립된 주요 콘텐츠 영역을 정의’하기 위해 사용하는 태그 는?

<section>

6. <hr> 태그에 사용할 수 있는 속성이 아닌 것은?

<href>

9. 웹 브라우저에 ‘<태그>’와 같이 출력되도록 바르게 작성한 것은?

< 태그 >

10. HTML 기본 태그 중 아래첨자를 표시하기 위해 사용하는 태그는?

<sub>

11. <a> 태그의 타깃(target) 속성값으로 사용하지 않는 것은?

<_non>

12. <audio> 태그의 속성에 대한 설명으로 옳지 않은 것은?

<preload>
